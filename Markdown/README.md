**Markdown 是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。**
_（因为是纯文本，所以只要支持 Markdown 的地方都能获得一样的编辑效果。例如：github/csdn/简书等）_

接下来简单描述一下语法：

### 1. 标题

> 标题使用`# title`来标记，`#`与内容保留一个空格，否则不生效；
> 多一个`#`代表降一级（即字号小一级），总共有 6 级；如：

```
# 这是一级标题
## 这是二级标题、
...
###### 这是六级标题
```

效果如下：

- # 这是一级标题

* ## 这是二级标题、

- ### 这是三级标题

* #### 这是四级标题

- ##### 这是五级标题

* ###### 这是六级标题

---

### 2. 字体

- 斜体

  > 斜体的内容需要使用一对下划线`_斜体_`或者一对星号`*斜体*`包裹起来即可（部分编辑器强制使用下划线），如果内容中含有相关标识符需要使用\进行转义与编程语言相似;

  例如：`_字体斜体_`

  效果：_字体斜体_

* 加粗

  > 加粗的内容需要使用两对星号`**加粗**`或者两对下划线`__加粗__`包裹起来即可（部分编辑器强制使用星号），如果内容中含有相关标识符需要使用\进行转义与编程语言相似;

  例如： `**字体加粗**`

  效果：**字体加粗**

- 斜体加粗

  > 斜体加粗的内容需要使用两对星号和一对下划线混合`**_斜体加粗_**`包裹起来即可；

  例如： `**_斜体 \*\_\* 加粗_**`

  效果：**_斜体 \*\_\* 加粗_**

* 删除线

  > 增加删除线的内容需要使用两对对波浪号`~~删除线~~`包裹起来即可

  例如：`~~字体删除线~~`

  效果：~~字体删除线~~

---

### 3. 引用

> 引用，就是描述内容标记为一个模块;

例如：

```
  > 这是一段详细的描述...
  >
  > > 这是描述的内部的引用...
  > >
  > > > > > > > 这是描述的内部的引用...
```

效果如下：

> 这是一段详细的描述...
>
> > 这是描述的内部的引用...
> >
> > > > > > > 这是描述的内部的引用...

---

### 4. 分割线

> 一条浅灰色的实线，使用三个或更多星号`***`或者三个或更多中线`---`进行标识,部分编辑器强制变更，依据当前位置上下文环境；

例如：

```
    ***
    ---
```

效果如下：

---

---

### 5. 图片

> 图片加载至支持使用网络图片,使用`![desc](image-url)`,和超链接有些相似;

例如：

```
![网络图片](http://inews.gtimg.com/newsapp_match/0/359390627/0)
```

效果如下：

![网络图片](http://inews.gtimg.com/newsapp_match/0/359390627/0)

---

### 6. 超链接

> 使用文字描述，支持点击进行跳转，使用`[tag](url)`,和图片有些相似;

例如：

```
[点击跳转](http://inews.gtimg.com/newsapp_match/0/359390627/0)
```

效果如下：

[点击跳转](https://typora.io/)

---

### 7. 列表

- ##### 有序列表

  > 有序列表：使用 `1.`,`2.`,`3.`...排下去,注意序号和内容之间需要空格分隔，例如：

  ```
  1. title
  2. font
  3. image
  4. list
  ```

  效果：

  1.  title
  2.  font
  3.  image
  4.  list

- ##### 无序列表

  > 无序列表: `+`,`-`,`*`...排下去，不需要排序，部分编辑器会自动强制，注意序号和内容之间需要空格分隔，例如：

  ```
  - 超链接
  * 代码
  - 逻辑条件
  * 引用
  ```

  效果：

  - 超链接

  * 代码

  - 逻辑条件

  * 引用

---

### 8. 代码

> 代码样式有两种：单行代码和多行代码(两者的区别就是多行样式中区分间距和换行，保留代码块的样式，而单行则忽略);
> 单行代码：使用一对反向单引号`` `code` ``包裹起来;

```
`This is a example`
```

> 效果如下：

`This is a example`

> 多行代码：使用三对反向单引号包裹起来;
> 保留样式的```效果如下(这种模式下支持多语言：c，java，php，js，python...几乎所有开发语言)：

````
```java // 这里指定代码的语言
  public static void main(String[] args){
      System.out.print("Hello world !")
  }
````

> 效果如下：

```java
  public static void main(String[] args){
     System.out.print("Hello world !")
  }
```

### 9. 表格

> 表格使用 `| tab | tab | tab |...|`声明表头字段
> 表格使用 `| :-- | :-: | --: |...|`申明表单中对应列中的内容对其方式,冒号在左侧即左对齐，在右侧即右对齐，在两侧即居中；
> 使用 `|value|value|value|...|`插入数据，例如：

```
| name | age | gender | user_info     |
| :--- | --: | :----: | :------------ |
| w    |  18 |   男   | 高 de 一批... |
| x    |  10 |   女   | 富 de 一批... |
| l    |  28 |   男   | 帅 de 一批... |
```

效果如下：

| name | age | gender | user_info     |
| :--- | --: | :----: | :------------ |
| w    |  18 |   男   | 高 de 一批... |
| x    |  10 |   女   | 富 de 一批... |
| l    |  28 |   男   | 帅 de 一批... |
